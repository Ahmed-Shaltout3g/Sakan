## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```sh
   git clone https://ahmed-shaltout3g.github.io/Sakan/.git
   cd Educational_platform


2. Install dependencies:
   ```sh
   npm install

3. Start the development server:
   ```sh
   npm start

## Usage
